# Сложно сосредоточиться

Статическая адаптивная страница по макету Яндекс Практикума. Проект посвящён причинам, по которым сложно концентрироваться, и простым способам вернуть внимание.

## Демо

Опубликованная версия: **https://thekostvk.github.io/slozhno-sosredotochitsya-fd/**

## Технологии

- **HTML5** — семантическая разметка страницы
- **CSS3** — адаптивная вёрстка, Grid Layout, CSS-переменные
- **БЭМ** — методология именования классов
- **JavaScript** — переключение цветовых тем
- **GitHub Pages** — публикация проекта

## Функциональность

- Адаптивная вёрстка для мобильного, планшетного и десктопного экранов
- Переключение тем: светлая, тёмная и автоматическая
- Поддержка системной темы через `prefers-color-scheme`
- Фиксированные фоновые изображения для разных тем
- Галерея изображений с ленивой загрузкой

## Структура проекта

- `index.html` — основная HTML-разметка
- `styles/` — CSS-файлы проекта
- `scripts/script.js` — логика переключения тем
- `images/` — изображения, иконки и фоны
- `fonts/` — локальные шрифты IBM Plex Mono

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Локальный просмотр

Проект статический, поэтому его можно открыть напрямую через `index.html` или запустить простой локальный сервер:

```bash
python3 -m http.server 4173
```

После запуска страница будет доступна по адресу:

```text
http://localhost:4173/
```

## Публикация

Для публикации на GitHub Pages используется пакет `gh-pages`.

```bash
npm run deploy
```
